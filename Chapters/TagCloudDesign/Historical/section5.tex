See Figure \ref{fig:selectmapping}, Figure \ref{fig:overview}  and Figure \ref{fig:filter} for examples on how to achieve this within a tag cloud implementation.

	\item Overview
	\item Zoom
	\item Filter
	\item Details-on-demand
	\item Relate
	\item History
	\item Extract
\end{enumerate}
